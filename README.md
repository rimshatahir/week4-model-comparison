# Week 4 Model Comparison

This project compares multiple AI models for analyzing cybersecurity alerts using an automated n8n workflow.

## Models Used
- HuggingFace Sentiment Analysis
- HuggingFace Zero-Shot Classification
- HuggingFace Named Entity Recognition
- Groq LLM

## Workflow
1. Retrieve alerts from Airtable
2. Send alerts to multiple AI models
3. Extract model outputs
4. Merge results
5. Update Airtable with predictions

## Technologies
- n8n
- Airtable
- HuggingFace API
- Groq API
